# Deploy Backend (Apps Script)

Deploy the Google Apps Script backend. $ARGUMENTS

## Steps

1. **Push code**: Run `cd apps-script && clasp push` to upload Code.js to Google Apps Script.

2. **Deploy**: Run `clasp deploy -i AKfycbz3DmnPOLstWmOmJs4nzDQn42XXWe0E2ujLpmfo4e4WZFkInXxUdeL8-W0SImYj9EQj -d "description"` where description summarizes the changes.

3. **Verify**: Confirm deployment ID is returned.

4. **Report**: Show the deployment result.
